# バイオリンQ&A相談室

LINE風のトーク画面で、バイオリンの悩み200問への回答（バイオリニスト・研究マニア・身体専門家の座談会）を読むための静的Webアプリ。

## データの流れ

- 原稿: `~/qa300-book-2026/chapters_v2/chNN/qNNN.json`（書籍の真実源。ここは書き換えない）
- `scripts/build-data.mjs` が原稿を読み、次の変換だけを行って `data/questions.js` を生成する
  - 相談者の実名（M.K.さん等）を「あなた」「お子さん」へ匿名化
  - 出典の内部ファイル名を読者向けラベル（【書籍】【研究論文】等）へ整形
  - どうしても直したい箇所だけ `OVERRIDES` で個別置換（書籍原文は温存）
- 回答本文は書籍の座談会をそのまま会話順に再生する（機械的な言い換えはしない）

## 更新

原稿JSONを更新したあと、次を実行する。

```bash
node scripts/build-data.mjs
```

キャッシュ対策として `index.html` の `?v=` を更新してからコミットする。

GitHub Pagesで `main` ブランチのルートを公開する（リモート `v2` = violin-qa-chat-v2）。
